# 程序截图目录

本目录用于存放 OpenClaw Files 应用程序的截图，以便在 GitHub README 中展示。

## 需要的截图

请准备以下截图文件（PNG 格式推荐）：

1. **main-interface.png** - 主界面截图
   - 显示三栏布局
   - 包含左侧资源树、中间文档预览、右侧分析面板
   - 展示应用的核心功能

2. **theme-switcher.png** - 主题切换功能截图
   - 显示顶部工具栏的主题选择器
   - 展示多套主题选项
   - 体现界面的美观性

3. **file-organization.png** - 文件整理过程截图
   - 显示分析结果和整理建议
   - 展示操作确认界面
   - 体现智能分析能力

## 截图建议

- 使用清晰的分辨率（推荐 1920x1080 或更高）
- 确保文字清晰可读
- 选择有代表性的文件夹作为示例
- 可以适当添加标注或说明
- 保持截图风格一致

## 更新 README

添加截图后，GitHub 会自动在 README 中显示。如果截图未显示，请检查：

1. 文件名是否正确
2. 文件是否在正确的目录
3. README 中的图片路径是否正确
